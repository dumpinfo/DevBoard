1 - Presentation
----------------
Les bibliothèques Proteus que je mets à disposition ici incluent :
- des composants et des symboles utilisables dans le logiciel ISIS
- des empreintes de composants utilisables dans ARES
- des modèles 3D au format 3DS utilisables dans le visualiseur 3D de Ares.
Copyrights: SONELEC-MUSIQUE, Remy Mallard - 2007


2 - Configuration requise
-------------------------
- Proteus V7.x, les versions précédentes de Proteus ne sont pas supportées.


3 - Installation 
-------------------------------------------------------------------
- Placer les librairies rm_sch_*.lib dans le répertoire par défaut LIBRARY 
de Proteus, ou mieux, dans un répertoire de votre choix qui devra être 
spécifié dans la fenêtre des options des chemins d'accès aux librairies 
(dans Isis, menu System, Set Path, champ Librairy folders).
- Placer les librairies rm_pcb_*.lib dans le répertoire par défaut LIBRARY 
de Proteus, ou mieux, dans un répertoire de votre choix qui devra être 
spécifié dans la fenêtre des options des chemins d'accès aux librairies 
(dans Ares, menu System, Set Path, champ Librairy folders).
- Placer les modèles 3D *.3ds dans le répertoire par défaut LIBRARY 
de Proteus, ou mieux, dans un répertoire de votre choix qui devra être 
spécifié dans la fenêtre des options des chemins d'accès aux librairies 
(dans Ares, menu System, Set Path, champ Librairy folders).


4 - Evolutions des librairies
-----------------------------
Toute demande d'évolution, de modification ou de correction, sera prise en 
compte. Afin de gérer correctement l'évolution des composants, le paragraphe 
"Evolutions demandées" du fichier A_lire.txt (celui que vous lisez 
actuellement) comportera vos demandes. Les demandes traitées apparaitront dans
le paragraphe "Historique" de ce même fichier A_lire.txt.

Remarque : le suivi ne sera pas forcément assuré de façon systématique et dans 
des délais courts.
